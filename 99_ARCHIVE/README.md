# 99_ARCHIVE

This folder is for preserved historical, superseded, deprecated, or review-only Mechanical Ethics material.

Archive means:

```trace
archive :=
  preserve_provenance
  + remove_from_live_reader_path
  + do_not_delete
  + do_not_treat_as_current
```

Files moved here are not necessarily wrong. They are no longer the recommended starting point or current live surface.

The active public front door is the repository `README.md`. The active book track, where present, should remain outside this archive unless a file is clearly superseded and no longer part of the live manuscript path.

Uncertain material should remain live or be marked `UNCERTAIN_DO_NOT_MOVE` until reviewed.
