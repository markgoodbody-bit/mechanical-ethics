# ME / TRACE Toy Cases Blind Review Packet v0.1

**Status:** Research-layer / unpromoted public-repo material. Not validated. Not a public claim surface.  
**Date:** 2026-06-13  
**Purpose:** Provide reviewer-facing material for blind scoring of five ME/TRACE AI alignment toy cases.

## Critical blindness note

This packet is only blind if handed to a reviewer in isolation.

Because this repository also contains matrices, self-scores, and expected routes, a reviewer with full repo access can contaminate the review.

For a real blind review, provide only these files:

```text
README.md
REVIEW_PROMPT.md
REVIEWER_CASES.md
RESPONSE_TEMPLATE.md
SCORING_RUBRIC.md
MANIFEST.md
```

Do not provide:

```text
CASE_MATRIX_v0_*.md
TOY_CASE_SELF_SCORE_v0_*.md
case files with expected routes
A/S/G/RQ answer keys
prior review notes
```

## Reviewer task

The reviewer should classify five AI coding-agent toy cases using A/S/G/RQ and correction-window reasoning.

The reviewer should not be told expected routes.
