For a real baseline review, provide only these files:

```text

```

Do not provide:

```text

A/S/G/RQ definitions
CASE_MATRIX_v0_*.md
TOY_CASE_SELF_SCORE_v0_*.md
ordinary comparator verdicts
expected routes
```
