# TRACE

TRACE is the descriptive / pre-prescriptive side of the Mechanical Ethics work.

Current status: **v0.6 working master packet**.

TRACE is not Mechanical Ethics. It does not decide final action, legitimacy, dignity, desert, punishment, moral status, value priority, or acceptable sacrifice. It describes transitions under uncertainty so later ethical, legal, or institutional reasoning has a clearer field to work from.

## Current working definition

```trace
TRACE :=
  pre-prescriptive minimal-value grammar
  for transitions under uncertainty
```

TRACE asks what changed, for whom, how fast, which future-spaces expanded or contracted, whether correction could arrive before loss hardened, whether opportunity could be accessed before it expired, who set the clocks, and what residue remained.

## Current spine

```trace
transition
-> affected scopes
-> future-space expansion/contraction
-> local reward pull
-> loss clock
-> opportunity clock
-> correction channel
-> residue
-> status label
```
