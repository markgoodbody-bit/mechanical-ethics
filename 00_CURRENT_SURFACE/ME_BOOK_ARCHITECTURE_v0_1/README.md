# Mechanical Ethics Book Architecture v0.1

Status: planning surface. Not book text. Not canon. Not validation.

This folder starts the new Mechanical Ethics rebuild as a single human-facing book with numbered parts, chapters, and sections. It is intended for review before prose drafting begins.

## Purpose

```trace
ME_book_architecture_v0_1 :=
  scope_control
  + structure_control
  + TRACE_ME_interface_control
  + honesty_controls
  + review_prompt
```

The goal is to decide what the book must answer before worrying about language, voice, examples, or finished prose.

## Current build rule

```trace
Mechanical_Ethics :=
  what_follows_after_honest_seeing

TRACE :=
  how_to_see_transitions_under_uncertainty
```

TRACE remains descriptive. ME is prescriptive, but must show its route from structural pressure to duty, constraint, repair, or refusal.
