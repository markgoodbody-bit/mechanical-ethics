# Archive

This folder is reserved for historical, superseded, or internal-context material if it is later added.

Do not put material here merely because it exists.

Archive material should be included only if it helps public audit, source history, or claim discipline.

## Rules

- Archive does not equal canon.
- Archive does not equal validation.
- Archive does not control the public front door.
- Public users should be able to use the main repository without reading archive material.

## Possible future contents

- superseded test versions;
- demotion notes;
- public release candidates;
- review packets;
- historical context.

If archive material is added, include a short note explaining why it matters and what authority it does not have.
