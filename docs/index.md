# Correction Window Test

**Can the safeguard actually fix the error before it is too late?**

This is a v0.1 diagnostic tool.

It asks:

> If a system is wrong, can the affected person get correction before the consequence becomes hard to reverse?

## Constraints

This is not a governance framework, ethics theory, certification scheme, or AI-alignment solution.

It is a logic test for one failure mode: a claimed safeguard that cannot correct a harmful error before the consequence hardens.
